Job Title: Backend Engineer
Company: Vertex Cloud Systems
Location: Hybrid (San Francisco, CA)

About the Role:
Vertex Cloud Systems is looking for a Backend Engineer to design, build, and scale the
services that power our platform. You will work in a microservices architecture running
on AWS and Kubernetes, collaborating with frontend, DevOps, and product teams to ship
reliable, well-tested code.

Responsibilities:
- Design and implement RESTful and gRPC APIs using Python (FastAPI/Django) or Node.js
- Build and maintain scalable microservices deployed on Kubernetes
- Write unit and integration tests to ensure code quality (pytest, Jest)
- Optimize database queries and schema design (PostgreSQL, Redis)
- Set up and maintain CI/CD pipelines (GitHub Actions, Jenkins)
- Collaborate with DevOps on infrastructure as code (Terraform, Docker)
- Participate in code reviews and on-call rotation
- Monitor systems using Prometheus and Grafana

Required Qualifications:
- Bachelor's degree in Computer Science or equivalent practical experience
- 3+ years of professional backend development experience
- Strong proficiency in Python or Node.js
- Experience with relational databases (PostgreSQL, MySQL) and caching (Redis)
- Hands-on experience with Docker and Kubernetes
- Familiarity with cloud platforms, preferably AWS
- Solid understanding of REST API design and microservices architecture
- Experience with Git and collaborative development workflows

Preferred Qualifications:
- Experience with Kafka or other message queues
- Exposure to Terraform or other infrastructure-as-code tools
- Experience with GraphQL
- Contributions to open source projects
- Master's degree in Computer Science

What We Offer:
- Competitive compensation and stock options
- Hybrid work model with modern SF office
- Top-tier health benefits
- Annual conference and learning stipend
